关于本系统 / About ▾

系统性质
刚果（金）武装冲突态势感知交互地图。仅做公开信息的汇总与可视化呈现，不构成独立安全分析或预测。

数据来源 / Data Sources
ACLED（武装冲突地点与事件数据项目） · GDELT（全球事件、语言与语调数据库） · UCDP（乌普萨拉冲突数据项目） · ReliefWeb（联合国人道事务协调厅）
所有数据均归因于具名第三方来源，并标注验证状态。

数据更新 / Update Mode
内置数据覆盖 2020–2026 年刚果（金）26个省份的主要冲突事件。 爬虫工具（scripts/crawler.py）每日 08:00 从 GDELT 2.0 API 自动拉取最新冲突报道并存入本地数据库。更新日志可在右侧浏览栏中查询。

地图交互 / Map Usage
• 点击省份 → 查看该省冲突概况
• 点击标记 → 查看事件详情
• 右侧浏览栏 → 筛选、统计、时间轴、周报、快速报告
• Esc 关闭弹窗

技术架构 / Technical
Leaflet 地图引擎 · CARTO 底图 · DRC 省级行政区划数据来自 geoBoundaries。 浏览器直接打开运行，无服务器依赖。数据层与展示层严格分离，数据更新不影响系统稳定性。

〓 项目位置：project-2/africa_security_map.html
〓 爬虫入口：python scripts/crawler.py --days 7
